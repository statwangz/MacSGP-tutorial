=============================================================================
Multiple human colorectal cancer datasets generated by different technologies
=============================================================================

The human colorectal cancer datasets profiled by Visium, Visium HD, Xenium, and scFFPE-seq :cite:p:`oliveira2025high` are all available at 
the project website (https://www.10xgenomics.com/platforms/visium/product-family/dataset-human-crc)
and the GEO DataSets database with accession number `GSE280318 <https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE280318>`_.
