===========================================================
Visium kidney cancer dataset at the tumour-normal interface
===========================================================

The kidney cancer dataset profiled by 10x Visium and paired scRNA-seq data :cite:p:`Li2022a` are 
available to download as h5ad objects in Mendeley Data (https://doi.org/10.17632/g67bkbnhhg.1). We used 
a sample with the donorid 'PD47171' in our analysis.
